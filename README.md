# Astock Skills

三款 Claude Code 技能，覆盖 A 股研究全流程：数据获取 → 基本面分析 → 交易决策。**用约束对抗随意——每层有不可跳过的检查点、强制反方论证、可证伪的跟踪条件，防止"看到什么就信什么"。**

## 技能列表

| 技能 | 触发 | 做什么 |
|:---|:---|:---|
| `tushare` | "查走势""拉财报""导出行情" | 数据获取、清洗、导出 |
| `stock-analysis` | "分析下XX""怎么看XX" | 三层递进基本面分析 |
| `stock-trading` | "XX交易分析""该买吗""还能拿吗" | 波段交易技术面分析 |

## 协作流程

```
用户提问
  ├─ "分析下XX" → stock-analysis（三层递进基本面）
  │    └─ 需要数据 → tushare（行情/财务/估值）
  │    └─ "还能拿吗" → stock-trading（技术面/资金面）
  │
  ├─ "XX交易分析" → stock-trading（纯交易面）
  │
  └─ "拉板块数据/查行情/导报表" → tushare（纯数据）
```

## 个股分析框架

```
第一层  定量初筛  →  行业坐标系 + 同业扫描 + 杜邦拆解
第二层  业务拆解  →  收入结构 + 技术路线 + 客户集中度 + 反方检查
第三层  风险定价  →  结构性风险 + 资产质量 + 周期位置 + 估值判断
第四层  持续跟踪  →  可证伪观察点 + 退出条件
```

## 个股交易框架

```
前置过滤  →  大盘环境 + 板块热度 + 流动性 + 事件日
时间框架  →  周线定方向 → 日线定位置 → 60分钟定时点
技术面    →  个股分类 + 量价四象限 + 均线系统
资金面    →  聪明钱 vs 杠杆资金
盈亏比    →  入场方案择优 → 持有/空仓决策
```

## 安装

```bash
# 复制到任意 Claude Code 项目的 .claude/skills/ 目录
cp -r skills/* /your-project/.claude/skills/
```

### 依赖

- **tushare**: Python 3.7+, `tushare` 包, `TUSHARE_TOKEN` 环境变量（[注册获取](https://tushare.pro/register)）
- **stock-analysis**: Python 3.7+, `akshare` 包。中证行业分类CSV已随技能打包在 `references/` 中
- **stock-trading**: 无额外依赖，大盘环境判断复用 `../stock-analysis/scripts/akshare_utils.py`。数据字典为顶层 `references/数据字典.md`

## 设计原则

1. **数据先行** — 不拉数据不做结论，禁止推测
2. **框架约束** — 每层有明确的"不可跳过"步骤，防止分析随意性
3. **基本面与交易面分离** — 两个框架独立运作，互不引用对方的指标
4. **按需加载** — 技能不触发不占上下文，比 memory 方式更高效
