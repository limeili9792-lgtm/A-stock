# 数据字典

## 一、stock-analysis 字段映射

### 第一层：定量初筛

| 分析步骤 | 数据源 | 接口 | 关键字段 | 状态 |
|:---|:---|:---|:---|:---|
| 行业分类链 | 中证CSV | `references/中证行业分类.csv` | 证券代码, 证券简称, 一级→四级行业代码及名称 | ✅ |
| 同业Top10(市值/PE) | Tushare | `daily_basic` | `ts_code, total_mv, pe_ttm` | ✅ |
| 杜邦拆解 | AKShare | `scripts/akshare_utils.py` → `get_financial_summary()` | ROE, 净利率, 毛利率, 资产负债率, 权益乘数, ROA | ✅ |

### 第二层：业务拆解

| 分析步骤 | 数据源 | 接口 | 关键字段 | 状态 |

| 收入结构 | 年报PDF | `scripts/extract_annual_report.py` → `extract_section("revenue_breakdown")` | 分产品营收、毛利率、同比 | ✅ |
| 收入结构(兜底) | WebSearch | — | 同上 | ✅ |
| 技术对比 | WebSearch | — | 技术路线、自给率 | ✅ |
| 客户集中度 | 年报PDF | `extract_section("customer_concentration")` | 前五大客户销售额、占比 | ✅ |
| 利润质量拆解 | AKShare | `scripts/akshare_utils.py` → `get_profit_quality()` | 净利率桥接(毛利率/费用率/非主营比重/缺口)、CFO/NP比率、增速一致性 | ✅ |
| 利润质量(兜底) | WebSearch | — | 毛利率、费用率、CFO、净利润 | ✅ |

### 第三层：风险定价

| 分析步骤 | 数据源 | 接口 | 关键字段 | 状态 |
|:---|:---|:---|:---|:---|
| 结构性风险 | WebSearch | — | 行业趋势、政策变化 | ✅ |

### 第四层：跟踪

| 分析步骤 | 数据源 | 接口 | 关键字段 | 状态 |
|:---|:---|:---|:---|:---|

## 三、数据源汇总

| 数据源 | 接口 | 频率限制 | 适用场景 | 状态 |
|:---|:---|:---|:---|:---|
| 中证CSV | 本地文件 | 无 | 行业分类 | ✅ |
| AKShare `abstract_ths` | `get_financial_summary()` | 无 | 杜邦拆解、财务摘要 | ✅ |
| AKShare `benefit/cash_new_ths` | `get_profit_quality()` | 无 | 季度利润质量拆解 | ✅ |
| AKShare `index_daily` | `get_index_status()` / `get_index_volume_status()` | 无 | 大盘环境定量+量能 | ✅ |
| Tushare `daily` | MCP | 200次/分钟 | 日线行情、量价 | ✅ |
| Tushare `daily_basic` | MCP | **5次/天** | PE/PB/市值/换手率 | ✅ |
| Tushare `weekly` | MCP | 200次/分钟 | 周线趋势 | ✅ |
| 年报PDF | `extract_section()` | 巨潮限速 | 分业务/客户/担保/存货/应收 | ✅ |
| WebSearch | — | — | 行业信息、资金面、业绩兜底 | ✅ |
| Tushare `moneyflow` | MCP | — | 资金流向 | ❌ 40203 |
| Tushare 财务三大表 | MCP | — | income/balancesheet/cashflow | ❌ 40203 |

## 四、不足项

| # | 不足项 | 影响 | 优先级 |
|:---:|:---|:---|:---:|
| 1 | `daily_basic` 5次/天 | 批量同业PE对比卡脖子 | 中 |
| 2 | `moneyflow` 无权限 | 资金面非结构化、滞后 | 中 |
| 3 | 财务三大表无权限 | AKShare已覆盖核心字段，影响小 | 低 |
| 4 | 交易框架大盘数据源不一致 | Tushare指数返回空→已用AKShare替代 | 低 |
